# Play Store Screenshots

Place your store screenshots here before uploading to Play Console.

## Requirements

- **Phone**: Minimum 2, recommended 8. At least one must be 320–3840 px on the short side.
- **7-inch tablet** (optional): Min 2. Same short-side range.
- **10-inch tablet** (optional): Min 2. Same short-side range.

## Suggested sizes

| Device   | Resolution (portrait) |
|----------|------------------------|
| Phone    | 1080 x 1920 px         |
| 7" tab   | 1200 x 1920 px         |
| 10" tab  | 1600 x 2560 px         |

## How to capture

1. Run the app on a device or emulator: `flutter run --release`
2. Use Device Preview or an emulator to capture key screens (home, gameplay, high score, settings if any).
3. Save files here with clear names (e.g. `phone_1_gameplay.png`, `phone_2_menu.png`).
